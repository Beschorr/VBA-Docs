# ChartCategory.Parent property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ A variable that represents a **[ChartCategory](Excel.chartcategory.md)** object.


## Property value

**Object**
