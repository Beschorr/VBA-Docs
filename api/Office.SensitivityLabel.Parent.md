# SensitivityLabel.Parent property (Office)

Gets the **Parent** object for the **SensitivityLabel** object. Read-only.


## Syntax

_expression_.**Parent**

_expression_ An expression that returns a **[SensitivityLabel](Office.SensitivityLabel.md)** object.


## Return value

Object
