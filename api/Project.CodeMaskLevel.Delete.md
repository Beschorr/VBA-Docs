# CodeMaskLevel.Delete method (Project)

Deletes the **CodeMaskLevel** object from a **CodeMask** collection.


## Syntax

_expression_.**Delete**

_expression_ A variable that represents a [CodeMaskLevel](./Project.CodeMaskLevel.md) object.
