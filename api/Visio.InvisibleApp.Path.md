# InvisibleApp.Path property (Visio)

Returns the drive and folder path of the Microsoft Visio application. Read-only.


## Syntax

_expression_.**Path**

_expression_ A variable that represents an **[InvisibleApp](Visio.InvisibleApp.md)** object.


## Return value

String
