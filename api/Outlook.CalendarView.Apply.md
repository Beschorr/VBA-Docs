# CalendarView.Apply method (Outlook)

Applies the **[CalendarView](Outlook.CalendarView.md)** object to the current view.


## Syntax

_expression_.**Apply**

_expression_ A variable that represents a [CalendarView](Outlook.CalendarView.md) object.
