# MediaFormat.Volume property (PowerPoint)

Gets or sets the volume of the media. Read/write.


## Syntax

_expression_. `Volume`

 _expression_ An expression that returns a [MediaFormat](PowerPoint.MediaFormat.md) object.


## Return value

Single
