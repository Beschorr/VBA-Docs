# XlSortOn enumeration (Excel)

Specifies the parameter on which the data should be sorted.



|Name|Value|Description|
|:-----|:-----|:-----|
| **xlSortOnCellColor**|1|Cell color.|
| **xlSortOnFontColor**|2|Font color.|
| **xlSortOnIcon**|3|Icon.|
| **xlSortOnValues**|0|Values.|
