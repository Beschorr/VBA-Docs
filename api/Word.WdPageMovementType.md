# WdPageMovementType enumeration (Word)

Specifies the page movement type.

|Name|Value|Description|
|:-----|:-----|:-----|
| **wdVertical**|1|Document page movement type vertical.|
| **wdSideToSide**|2|Document page movement type side-to-side.|
